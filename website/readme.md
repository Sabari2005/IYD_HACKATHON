## "IYD HACKATHON - WEB UI"

## Installation

### Prerequirements
  - `python3.11`

### Installation steps

  ```
    https://github.com/Sabari2005/IYD_Hackathon.git
    cd IYD_Hackathon
    cd website
  ```
  ```
    pip install -r requirements.txt
  ```
  - run the server
  ```
  python app.py
  ```
  - Open ` http://127.0.0.1:8000` in your browser
